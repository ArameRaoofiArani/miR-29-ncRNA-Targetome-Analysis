# miR-29-ncRNA-Targetome-Analysis
This repository contains scripts used for the analysis of CLIP-seq and RNA-seq data in the dissertation: "Identification of MicroRNA-29 Noncoding RNA Targetome in Keratinocytes"
## Contents
- Differential expression analysis (DESeq2)
- Filtering pipeline for candidate ncRNAs
- Correlation analysis with miR-29-associated genes
- Visualization (bubble plots, expression plots)

## Data sources
- CLIP-seq dataset (Thiagarajan et al., 2025)
- RNA-seq dataset: GSE270877
- GTEx Portal (Release V10)

## Notes
Raw data are publicly available from the respective sources. This repository contains processed outputs and analysis scripts only.
